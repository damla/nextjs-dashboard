# NextJS Dashboard

## Technologies

- NextJS
- Schadcn/ui
- Tanstack Table
- Husky
- Prettier
- Eslint
- Tailwindcss
- Prisma ORM
- Postgres
- Supabase

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
